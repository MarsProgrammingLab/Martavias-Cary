* Project 1: Asteroids Game Application


A single-player game built with Java & Eclipse IDE.

This project was developed in my Object-Oriented Programming

course at UCF and is a re-creation of the Atari 7800 classic

Asteroids. The objective of the game is surviving and acheiving

the highest score possible by firing projectiles at

incoming asteroids.


* Project 2: Maze-Runner Application


A simulator built with Java using NetBeans.

This was a simulator developed in my Data Structures

& Algorithms course at UNC Charlotte.

The goal of this project is to traverse a

procedurally generated maze until the correct path to the

goal (cheese) is found.


* Project 3: Scientific Calculator using Python

A simple calculator written in Python.

This project was developed in my Software

Engineering Course at UNC Charlotte.

This is a calculator that can perform multiplication,
