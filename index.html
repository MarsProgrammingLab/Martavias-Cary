* Project 1: Asteroids Game Application


A single-player game built with Java + Eclipse IDE.

This project was developed in my object-oriented programming course at UCF and

is a recreation of the Atari 7800 classic Asteroids,

with the objective of surviving and acheiving the highest score possible

by firing projectiles at incoming asteroids.


* Project 2: Maze-Runner Application


A simulator built with Java using NetBeans. This was a simulator developed in my Data Structures & Algorithms course at UNC Charlotte.

The goal of this project is simple: To traverse a procedurally generated maze until the optimal path to the goal (cheese) is found.
